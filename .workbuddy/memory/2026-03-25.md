## 修复 giscus 初始深色模式不生效问题

- 根因：原先 `data-theme="preferred_color_scheme"` 是静态值，giscus iframe 加载时独立判断系统配色，忽略了 Blowfish 已通过 JS 强制加上的 `dark` class
- 修复方案：改为 JS 动态注入 giscus script；`DOMContentLoaded` 时 Blowfish 的 appearance.js 已同步执行并写好 `dark` class，此时读取 class 状态写入 `data-theme`，彻底消除时序竞争

- 创建 layouts/partials/comments.html，嵌入 giscus 脚本（repo: wqchnggyu/Comments）
- 深浅色适配：初始主题用 `preferred_color_scheme`，切换时通过 postMessage 发给 giscus iframe 同步新主题
- 监听点：`appearance-switcher` / `appearance-switcher-mobile` 点击，以及系统 `prefers-color-scheme` 变化
- params.toml 补回 showComments = true（layouts 目录之前被清空）
